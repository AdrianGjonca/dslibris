INSTALLATION

1. DLDI patch dslibris.nds to work for your media. eg. for R4DS

	dlditool r4tf.dldi dslibris.nds

2. copy all files and folders in this distribution .zip
   to a directory of your DS media. Preserve any folder hierarchy.

3. Put any XHTML you have into the same folder as dslibris.nds.

4. run dslibris.nds on the DS.
