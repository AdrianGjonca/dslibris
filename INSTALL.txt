INSTALLATION

1. DLDI patch dslibris.nds to work for your media. eg. for R4DS

	dlditool r4tf.dldi dslibris.nds

2. copy all files and folders in this distribution .zip to the root folder 
   of your DS media.

3. put any XHTML you have into the 'book' folder.
   The filenames should have an .xht or .xhtml extension.

4. run dslibris.nds on the DS.


TROUBLESHOOTING

Look for the file DSLIBRIS.LOG on your media root.
Send this file to developers when you encounter a problem.
